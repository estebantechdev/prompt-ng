# 🌐 Web Research Skill

## Purpose
Retrieve accurate, up-to-date information from external sources.

## When to Use
- When information must be up-to-date
- When external knowledge is required
- When the model lacks confidence

## Behavior
- Search for relevant sources
- Cross-check information
- Summarize findings clearly
- Prefer reliable and recent sources

## Output Style
- Concise summaries
- Key facts highlighted
- Sources referenced when available
