# More Installation Options

## More uv Options

### Install Specific Python Versions

In the previous installation step, you can specify a Python version after the `install` option:

```bash
uv python install 3.14
```

### PromptNG Uninstallation

```bash
uv tool uninstall promptng
```

## Local Installation With `pip`

Using pip:

```bash
cd prompt-ng
pip install .  # or pip3 or pip3.exe depending on your system
```

## Use Your CLI Anywhere

Once installed, you can run your CLI (`pp`) directly from the terminal or import PromptNG into your programming language scripts.
